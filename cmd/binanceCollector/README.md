# Binance 数据采集器

## 使用方法

```toml
APIKey = ""
SecretKey = ""
```

1. 填写并保存以上内容到 `binance.toml` 文件中。
1. 运行 `main.go` 程序。

Binance 的历史交易数据会保存到 `binance.sqlite3` 文件中。
